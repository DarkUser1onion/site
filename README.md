# Документация веб-сайта “Aircrack Handshake”

## Информация о проекте

Сайт **Aircrack Handshake** представляет собой учебный веб-ресурс, посвящённый взлому Wi-Fi сетей с защитой WEP/WPA/WPA2 с использованием инструмента Aircrack-ng. Основной акцент сделан на понимание процесса 4‑way handshake и практическое применение команд для захвата и брутфорса handshake.

Сайт разработан в рамках изучения HTML5 и CSS3 и не использует JavaScript, что соответствует требованиям учебного проекта.

**Цели веб-сайта:**
- Предоставить начинающим специалистам по кибербезопасности понятную информацию о принципах работы WPA2 handshake.
- Показать практические примеры использования инструментов Aircrack-ng (airodump-ng, aireplay-ng, aircrack-ng).
- Сформировать культуру этичного хакинга, подчеркнув важность тестирования только на своих сетях.

**Задачи веб-сайта:**
- Предоставить теоретический материал о 4‑way handshake.
- Продемонстрировать пошаговые инструкции по захвату handshake.
- Содержать контактную форму для обратной связи.
- Обеспечить адаптивный доступ с любых устройств.

**Целевая аудитория:**
- **Профессии:** студенты IT-специальностей, начинающие пентестеры, системные администраторы, интересующиеся кибербезопасностью.
- **Местоположение:** Россия и русскоязычные пользователи.
- **Возраст:** от 12 лет, но желательно от 16
- **Пол:** любой.
- **Язык:** русский.
- **ОС:** Linux.
- **Браузеры:** современные версии на базе Chromium 140+ или Firefox 143+
- **Уровень пользователей:** beginner, medium.

## Логическая структура сайта

Сайт имеет иерархическую структуру, состоящую из четырёх основных страниц:

1. **Главная страница (index.html)** - теория: объяснение WPA2 handshake, схема обмена пакетами, пример вывода airodump-ng.
2. **Практика (practice.html)** - пошаговая инструкция с реальными командами для захвата handshake и пояснениями к флагам.
3. **О проекте (about.html)** - информация о целях создания сайта, используемых инструментах и всякие полезные ссылки.
4. **Контакты (contacts.html)** - форма обратной связи и контактные данные (Telegram, GitHub).

Все страницы связаны единой навигацией в шапке и подвале.
